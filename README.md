# AI Friend Chatbot

A friendly AI chatbot built with Streamlit and powered by the DeepSeek API. This chatbot is designed to be warm, empathetic, and engaging in conversation.

## Features

- Modern, user-friendly interface
- Real-time chat experience
- Persistent chat history
- Clear chat functionality
- Responsive design

## Setup

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
streamlit run app.py
```

3. Open your web browser and navigate to the URL shown in the terminal (typically http://localhost:8501)

## Usage

1. Type your message in the chat input box at the bottom of the screen
2. Press Enter or click the send button to send your message
3. The AI will respond with a friendly and engaging message
4. Use the "Clear Chat" button in the sidebar to start a new conversation

## Note

This application uses the DeepSeek API for generating responses. Make sure you have a valid API key configured.
